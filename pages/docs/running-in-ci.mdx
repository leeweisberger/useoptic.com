# Running Optic in CI

> Connecting `optic diff` to your CI Provider and Code Host makes it easy for every developer to review API changes and follow your team's standards.

**Visual API Changelogs on Pull Requests:**
Large YAML diffs are hard to understand and review. Optic generates visual API changelogs to make code review more effective:
