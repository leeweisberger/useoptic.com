# Get started with Optic

APIs built with Optic are accurately documented, never ship breaking changes, and meet the high standards your consumers expect. Our [open source tools](https://github.com/opticdev/optic) make it easy for any team to adopt OpenAPI and work API-first.


**Install the Optic CLI**

## Create an accurate OpenAPI specs

- If you do not have an OpenAPI spec, Optic can help you **generate a spec in < 10 minutes.**
- If you already have an OpenAPI spec, Optic can **verify the spec is accurate.** If diffs are found passing Optic the  `--patch` flag will update the spec for you.
